CodeLive – Real-Time Code Editor

CodeLive is a lightweight, feature-rich real-time code editor built with vanilla JavaScript, inspired by CodePen. It is designed for intermediate developers to practice and demonstrate advanced front-end development concepts.

Features

Live preview for HTML, CSS, and JavaScript

Syntax highlighting using CodeMirror with multiple themes

Auto-run mode with debounced updates

Integrated console for logs, warnings, and errors

Automatic code persistence using localStorage

Horizontal and vertical layout switching

Fullscreen preview mode

Keyboard shortcuts for save and run actions

Pre-built templates for faster prototyping

Tech Stack

Vanilla JavaScript (ES6+)

CodeMirror 5

HTML5 and CSS3

LocalStorage API

postMessage API
